# 介绍

### 定位

antm.js Doc 是一个面向前端技术文档的框架, 以最简单、高效的方式构建技术文档。
支持`h1`和`h3`的`markdown`的标签形式， 即

```markdown
# 大标题

### 副标题
```

支持常见的文档搭建需求：

- `Markdown`语法扩展
- 自定义全局样式
- 全文搜索

后续将支持：

- 自定义主题
- 头部导航栏菜单
- 分辨率适配

### 功能特性

#### 支持 Markdown 语法扩展

antm.js Doc 基于`MarkdownItPlugin`进行 Markdown 语法的扩展

详情请参考[markdown 语法扩展](/#/markdown-expand)
